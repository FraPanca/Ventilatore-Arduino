# Ventilatore-Arduino

## Italiano
Un ventilatore regolabile tramite potenziometro.

Realizzazione di un ventilatore con una scheda Arduino UNO R3.
L'accensione e lo spegnimento avviene tramite bottone e la velocità può essere regolata mediante il potenziometro.
L'intensità della velocità verrà mostrata nel display BCD. 

### Componenti necessari
- UNO R3 Controller Board
- Fan Blade
- 3-6V Motor CC
- Button
- 1 Digit 7-Segment Display
- Potenziometro 10k
- NPN Transistor PN2222
- Diode Rectifier
- 3 resistori 220 Ohm
- Breadboard
- Breadboard jumpers

## English
A fan adjustable via potentiometer.

Implementation of a fan using an Arduino UNO R3 board.
Turning on and off is done via a button, and the speed can be adjusted using the potentiometer.
The speed intensity will be shown on the BCD display.

### Required components
- UNO R3 Controller Board
- Fan Blade
- 3-6V Motor CC
- Button
- 1 Digit 7-Segment Display
- 10k Potentiometer
- NPN Transistor PN2222
- Diode Rectifier
- 3 resistors 220 Ohm
- Breadboard
- Breadboard jumpers
